steps:- 
open git bash and "git clone https://github.com/rutujdeshpande/drupaltimezone.git".
copy "timezone_handler" folder in your drupal module folder. 
install the module 
goto main menu-> configuration -> sytem -> Timezone Form . you will be redirected to /admin/config/timezone_handler.
enter all field details and submit the form
goto /admin/structure/block/list and add timezone block in any section
goto homepage and check the required block
